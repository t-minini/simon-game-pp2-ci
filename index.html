Simon Game Test your memory and see how far you can go!

Game Rules

  • Start the Game
    Press the "Start" button to begin.
  • Watch Carefully
    The game will flash a sequence of colors. Remember the order.
  • Repeat the Sequence
    After the sequence stops, click the buttons to repeat the colors in the same order.
  • Level Up
    If you repeat the sequence correctly, the game adds one more color to the sequence for the next round.
  • Game Over
    If you make a mistake, "WRONG SEQUENCE!" will be displayed. Press "Reset" to try again.
  • Winning
    Continue as long as you can, trying to remember longer sequences each time. You win after getting 10 correct sequences.
